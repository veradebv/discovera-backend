## Description

Discovera backend API built with NestJS, Prisma, and JWT authentication. It provides book management, reviews, and user profile endpoints.

## API Endpoints

### Auth

- `POST /auth/register`
- `POST /auth/login`

### Users

- `GET /users/me` (protected)

### Books

- `POST /books` (protected, supports file upload `cover`)
- `GET /books`
- `GET /books/:id`
- `PATCH /books/:id` (protected)
- `DELETE /books/:id` (protected)

### Reviews

- `POST /books/:bookId/reviews` (protected)
- `GET /books/:bookId/reviews`
